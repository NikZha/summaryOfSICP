# 📚 Конспект по SICP (Structure and Interpretation of Computer Programs)

Добро пожаловать в мой конспект по легендарной книге «Структура и интерпретация компьютерных программ» (SICP).

Здесь собраны решения упражнений, заметки по теории, примеры кода и эксперименты с различными реализациями Scheme (CHICKEN, Racket) и даже сравнения с JavaScript.

## 🛠️ Инструменты

Для работы с кодом использую:

- **CHICKEN Scheme** (`csi`) — основной интерпретатор для проверки кода из SICP
- **Racket + DrRacket** — для отладки и визуализации процессов
- **VS Code** — для написания кода (с расширениями для Scheme)
- **Node.js** — для экспериментов с рекурсией и сравнения поведения

Подробнее о настройке и выборе инструментов: [tools/README.md](tools/README.md)

## 📖 Как читать конспект

1. **Упражнения** — решения с комментариями, часто в нескольких вариантах (рекурсивный, итеративный)
2. **Заметки** — объяснения ключевых концепций: рекурсивный vs итеративный процесс, аппликативный vs нормальный порядок, хвостовая рекурсия, ленивые вычисления
3. **Эксперименты** — сравнение поведения кода в Scheme и JavaScript, тесты на переполнение стека

## 🧠 Ключевые идеи, которые я усвоил

- **Рекурсивная процедура ≠ рекурсивный процесс**
- **Хвостовая рекурсия (TCO)** — это не оптимизация, а фундаментальное свойство Scheme, меняющее способ мышления
- **Аппликативный порядок** (Scheme) vs **нормальный порядок** (ленивые языки) — разница в количестве вычислений
- **Инструменты — это средство, а не цель** (после долгих танцев с Emacs и LispIDE)

## 🚀 Запуск примеров

### CHICKEN Scheme
```bash
csi -script chapter1/название упражнения/название упражнения.scm
```
Или:

```bash
node chapter1/название упражнения/название упражнения.js
```
